Справка

Главная

Окно с одной единственной формой ввода необходимо для аутентификации пользователя. Для просмотра раписания достаточно ввести номер групппы, но для возможности редактирования необходим ваш ID, который всегда можно спросить у бота.

Расписание

На странице находится полоса с номерами недель при нажатии на номер открывается расписание для соответствующей недели. По умолчанию при открытии страницы отображается текущая неделя.

Редактор

На странице радектора можно добавить, удалить, скрыть и изменить пары.
Расписание формируется из двух слоев:
1. Пары из официального расписания, которые не подлежат изменению.
2. Пользовательские пары. При редактировании пары из официального расписания создается ее пользовательская копия, и уже с ней можно совершать все выше описанные действия.
Два слоя пар нужны, чтобы можно было отслеживать изменения в официальном расписании, а также для сохранности пользовательского расписания.

Режим

"Официальное расписание" - пары из документов с расписанием с сайта МТУ, без каких-либо изменений. При данном режиме просмотра функция редактирования недоступна.
"Моё расписание" - режим в котором производится редактирование, отображает все пары с учетом произведенных изменений.

Вид

Здесь определяется какие сведения о парах отображаются в таблице редактора.

Уведомления

Включение/выключение уведомлений о предстоящих парах. Время уведомлений установить нельзя из-за технических особенностей работы бота в vk.
"Пары сегодня" - список предстоящих сегодня пар. Приходит с 3 до 6 утра.
"Пары завтра" - список пар на завтра. Приходит с 3 до 6 утра.
"Пары на неделю" - список пар по дням на всю неделю. Приходит с 3 до 6 утра воскресения.
"Первая пара" - присылает схему с аудиторей где проходит первая за день пара (вместе с названием пары)

Описание

Просто спросите расписание, как спрашиваете обычно у одногруппников.
В групповой беседе надо обращаться к боту "расписание" или "бот".

Бот может показать:

  • - следующие пары
  • - пары сегодня
  • - пары завтра
  • - пары послезавтра
  • - пары вчера
  • - пары в понедельник .. субботу
  • - текущую пару
  • - пару по номеру (первая..пятая)
  • - пары по дате ("1 ноября" или "1.11")
  • - пару по времени ("17:30")
  • - кто преподаватель на паре (по номеру, не по названию)
  • - какая идет неделя
  • - сколько прошло с начала семестра
  • - сколько осталось до конца семестра
  • - схему этажа с нужной аудиторией ⚠
  • - расписание экзаменационной сессии *выключено*
  • - отдельно экзамены *выключено*
  • - отдельно консультации *выключено*
  • - расписание зачетной сессии *выключено*
  • - календарь на декабрь или январь
  • - время начала и конца пар
  • - выдать краткую инструкцию
  • - сохраненный номер группы

Примеры сообщений:

  • "Какие завтра пары?"
  • v"Что у меня в пятницу в 17:00?"
  • "Расписание на 1 ноября"
  • "Кто лектор в четверг на первой паре?"
  • "Аудитория Г-301д"
  • "Где 1 пара во вторник?"⚠
  • "Какую группу ты запомнил?"

Замечания

Схемы этажей можно комментировать, если у вас есть относящаяся к схеме информация, обязательно напишите - всем будет интересно!

Во избежании блокировки добавлены различные ограничения:
- Бот отвечает только на понятные ему сообщения.
- Бот НЕ ПИШЕТ один и тот же ответ больше ОДНОГО раза.
- Бот может ответить с задержкой, но не больше 15 секунд.

Сообщить об ошибке или написать разработчику можно 2 способами:
1. Добавить к сообщению пометку "ошибка", "предложение", "разработчику".
2. Отправить письмо на timetable.bot@yandex.ru
